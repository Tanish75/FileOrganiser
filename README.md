# 📂 File System Analyzer & Organizer

A **command-line C++ tool** that scans, analyzes, and organizes messy folders into neat categories (PDFs, Images, Code, etc.). Built with **C++17** and the **`<filesystem>`** library.

---

## 🚀 Features

- **🔍 Recursive Scanning:** Goes through every subfolder to find all files.
- **📊 File Analysis:** Counts file types and displays the top 5 largest files.
- **🗂️ Smart Organization:** Automatically creates categorized folders and moves files into them.
- **⚡ Fast & Efficient:** Processes thousands of files in seconds.

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| **C++17** | Core programming language |
| **STL** | `std::vector`, `std::unordered_map`, `std::sort` |
| **`<filesystem>`** | Directory traversal, file operations |
| **Git** | Version control |
